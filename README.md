# Install Snap

## Step 1: Update Package Lists
```bash
sudo apt update
```
## Step 2: Install Snap
```bash
sudo apt install snapd -y
```

## Step 3: Enable the Snap service
```bash
sudo systemctl enable --now snapd
```
## Step 4: Verify the Installation
```bash
snap --version
```
